# 🗺️ MapifyMUJ

> Campus Navigation & Smart Student Dashboard for Manipal University Jaipur

MapifyMUJ is a responsive web application that helps students easily navigate campus buildings, view their timetable, and track attendance — all in one place.

---

## 🌐 Live Demo

👉 https://your-username.github.io/mapifymuj

---

## ✨ Features

### 🏠 Home Dashboard
- Personalized greeting based on time
- Displays **current/next class**
- Shows subject, faculty, room, and timing
- Attendance indicator for each subject
- List of upcoming classes

### 🏢 Campus Navigation
- Select building (AB1, AB2, AB3, LHC)
- Navigate through floors
- View all rooms in a structured grid

### 📅 Timetable
- Weekly timetable view
- Day-wise class breakdown
- Attendance overview for all subjects

### 📱 Responsive Design
- Mobile-friendly (app-like UI)
- Desktop-friendly (website layout)

## 🌐 Deploy on GitHub Pages

1. Push both files to a GitHub repo named `MapifyMUJ`
2. Go to **Settings → Pages**
3. Source: **Deploy from branch → main → / (root)**
4. Save — your app goes live in ~1 minute at:
   `https://yourusername.github.io/MapifyMUJ`
